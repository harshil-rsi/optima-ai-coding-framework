VB6 Example (End-to-End)

Use-case: Expose invoice lookup from VB6 module modBilling.GetInvoiceById(id as Long)

Prompt (Cursor):

# Build a call graph of public routines. Identify API-safe functions 
# (pure or constrained side-effects). Propose endpoints and schemas. 
# Output a table: Function | Inputs | Outputs | Side Effects | Endpoint | Risk.

OpenAPI Snippet:

paths:
  /invoices/{id}:
    get:
      summary: Get invoice by ID
      parameters:
        - name: id
          in: path
          required: true
          schema: { type: integer }
      responses:
        '200':
          description: OK
          content:
            application/json:
              schema: { $ref: '#/components/schemas/Invoice' }

Adapter Contract (C#):

public InvoiceDto GetInvoiceById(int id) {
    var result = Vb6Shim.Invoke("modBilling", "GetInvoiceById", id);
    return MapToDto(result);
}
