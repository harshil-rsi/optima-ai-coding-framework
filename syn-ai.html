Case Study · Insurance Software

Global policy administration vendor needed privacy-safe datasets to performance test a new microservices release and enable ML-based fraud scoring experiments—without copying production.

Scope
  • • 120+ entities (Policies, Claims, Endorsements, FNOL)
  • • Referential integrity across 1.6k FKs
  • • 5 TB synthetic volume for load tests
Approach
  • • MCP connectors to ERWin + Snowflake catalog
  • • Blueprinted loss ratios, seasonality, fraud patterns
  • • Output: SQL & Parquet with dbt seeds
Impact
  • • 42% faster performance test cycles
  • • 0 PII exposure for UAT/staging
  • • 18% lift in fraud model recall (A/B on synthetic)
